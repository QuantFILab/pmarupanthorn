### üñ•Ô∏è ‡∏ï‡∏±‡∏ß‡∏≠‡∏¢‡πà‡∏≤‡∏á‡πÇ‡∏Ñ‡πâ‡∏î Python (Empirical Estimation)

```python
import numpy as np
from scipy.stats import rankdata

def empirical_tail_dependence(x, y, u=0.05):
    n = len(x)
    ux = rankdata(x) / (n + 1)
    uy = rankdata(y) / (n + 1)

    lower = np.sum((ux <= u) & (uy <= u)) / np.sum(ux <= u)
    upper = np.sum((ux > 1 - u) & (uy > 1 - u)) / np.sum(ux > 1 - u)

    return lower, upper
```
